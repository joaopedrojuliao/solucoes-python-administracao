# Algoritmos de Apoio à Tomada de Decisão Administrativa 📊

Este repositório contém a resolução da **Lista de Exercícios 02** da disciplina de lógica de programação aplicada ao curso de **Administração**. O projeto foca no uso de estruturas condicionais (`if/elif/else`) e estruturas de repetição (`for`) para resolver problemas cotidianos de uma empresa.

## 🚀 Tecnologias Utilizadas
* **Python 3**
* **Jupyter Notebook** (Google Colab)

## 📋 Conteúdo do Projeto

O projeto está dividido em duas partes principais:

### Parte 1: Estruturas Condicionais (Tomada de Decisão)
Algoritmos desenvolvidos para automação de regras de negócio:
1.  **Classificação de Inadimplência:** Cálculo de status de fatura e juros com base nos dias de atraso.
2.  **Bônus de Performance:** Cálculo automático de premiação para vendedores conforme meta batida.
3.  **Prioridade de Atendimento:** Validação de critérios legais para filas prioritárias (idade e acessibilidade).
4.  **Viabilidade de Investimento:** Classificação de risco baseada no ROI (Retorno sobre Investimento).

### Parte 2: Estruturas de Repetição e Listas
Processamento de dados em lote:
5.  **Soma de Horas Extras:** Cálculo do total de horas a pagar para uma equipe de 10 funcionários.
6.  **Filtro de Estoque Crítico:** Sistema de alerta para reposição de itens abaixo do estoque mínimo.
7.  **Reajuste Salarial:** Aplicação de aumento percentual em massa em uma lista de salários.
8.  **Contador de Faltas:** Monitoramento de presença semanal de colaboradores.
